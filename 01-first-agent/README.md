# Article 1: Your First Autonomous Agent in 30 Minutes

Learn the fundamentals of building AI agents by creating a file reader that uses tools autonomously.

## 📖 Read the Article

**[Read on Medium →](https://medium.com/@rezarezvani/agent-builders-playbook-first-agent)**

## 🎯 What You'll Build

A simple but powerful agent that can:
- Read files from your filesystem
- Answer questions about file contents
- Decide which tool to use autonomously
- Handle errors gracefully

## 🛠️ Implementations

Choose your language:
- **[TypeScript Implementation](./typescript/)** (Recommended)
- **[Python Implementation](./python/)** (Full equivalent)

## 🧠 Key Concepts

This article teaches:
1. **Agent vs. API** - Understanding autonomy
2. **Tool Definition** - Describing capabilities to the agent
3. **Tool Implementation** - Executing tool calls
4. **Agent Loop** - Handling multi-step reasoning

## 💡 What You'll Learn

- Set up Claude Agent SDK from scratch
- Define and implement tools
- Build the agent orchestration loop
- Handle tool use responses
- Debug common errors

## ⏱️ Time Required

- **Setup:** 5 minutes
- **Reading + Coding:** 25 minutes
- **Experimentation:** 10+ minutes

## 📚 Next Steps

After completing this article:
1. Try the exercises in each implementation's README
2. Experiment with different questions
3. Add your own tools (list_files, write_file, etc.)
4. Move on to **Article 2: Documentation Generator**

## 🐛 Troubleshooting

**Common issues:**
- API key not found → Check `.env` file format
- Module not found → Run `npm install` or `pip install`
- File not found → Use paths relative to project root

See implementation READMEs for detailed troubleshooting.
